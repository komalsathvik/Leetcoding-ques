#lc 787 - Cheapest Flights within K stops
There are n cities connected by some number of flights. You are given an array flights where flights[i] = [fromi, toi, pricei] indicates that there is a flight from city fromi to city toi with cost pricei.

You are also given three integers src, dst, and k, return the cheapest price from src to dst with at most k stops. If there is no such route, return -1.

 

Example 1:


Input: n = 4, flights = [[0,1,100],[1,2,100],[2,0,100],[1,3,600],[2,3,200]], src = 0, dst = 3, k = 1
Output: 700
Explanation:
The graph is shown above.
The optimal path with at most 1 stop from city 0 to 3 is marked in red and has cost 100 + 600 = 700.
Note that the path through cities [0,1,2,3] is cheaper but is invalid because it uses 2 stops.
Example 2:


Input: n = 3, flights = [[0,1,100],[1,2,100],[0,2,500]], src = 0, dst = 2, k = 1
Output: 200
Explanation:
The graph is shown above.
The optimal path with at most 1 stop from city 0 to 2 is marked in red and has cost 100 + 100 = 200.
Example 3:


Input: n = 3, flights = [[0,1,100],[1,2,100],[0,2,500]], src = 0, dst = 2, k = 0
Output: 500
Explanation:
The graph is shown above.
The optimal path with no stops from city 0 to 2 is marked in red and has cost 500.
 

Constraints:

1 <= n <= 100
0 <= flights.length <= (n * (n - 1) / 2)
flights[i].length == 3
0 <= fromi, toi < n
fromi != toi
1 <= pricei <= 104
There will not be any multiple flights between two cities.
0 <= src, dst, k < n
src != dst

#approach Solution using Dijkstra's algo (modified)

Build the graph as an adjacency list from the given flights.

Use a priority queue to simulate a Dijkstra-like traversal, but instead of just (node, dist), we also keep the number of stops (k) in our state.

For each state (node, cost, stops) popped from the PQ:

If the number of stops exceeds k, skip it.

Otherwise, try to relax its neighbors. If traveling to a neighbor improves the cost, push (neighbor, newCost, stops+1) into the PQ.

The key detail is to use the current state’s distance (curr.dist) instead of the global best arr[u], because we’re exploring different paths with different stop counts. Using curr.dist ensures we’re working with the real cost of the path being expanded.

At the end, if the destination is still Integer.MAX_VALUE, it means it’s unreachable within k stops, otherwise return the cheapest cost.

Complexity
Time complexity:O(E * k log V)
Space complexity:O(E+V*k) ~ O(E+V),if k is small
#Code
class Solution {
    class Edge{
        int src;
        int dest;
        int wt;
        public Edge(int src,int dest,int wt){
            this.src=src;
            this.dest=dest;
            this.wt=wt;
        }
    }
    class Info implements Comparable<Info>{
        int v;
        int dist;
        int k;
        public Info(int v,int dist,int k){
            this.v=v;
            this.dist=dist;
            this.k=k;
        }
        @Override
        public int compareTo(Info i){
            return this.k-i.k;
        }
    }
    public void createGraph(ArrayList<Edge>[] graph,int[][] edges){
        for(int[] edge:edges){
            graph[edge[0]].add(new Edge(edge[0],edge[1],edge[2]));
        }
    }
    public int[] dijkstra(ArrayList<Edge>[] graph,int src,int k){
        int[] arr=new int[graph.length];
        Arrays.fill(arr,Integer.MAX_VALUE);
        arr[src]=0;
        PriorityQueue<Info> pq=new PriorityQueue<>();
        pq.add(new Info(src,0,0));
        while(!pq.isEmpty()){
            Info curr=pq.remove();
            if(curr.k>k)continue;
            for(int i=0;i<graph[curr.v].size();i++){
                Edge e=graph[curr.v].get(i);
                int u=e.src;
                int v=e.dest;
                int wt=e.wt;
                if(curr.dist+wt<arr[v] && curr.k<=k){
                    arr[v]=curr.dist+wt;
                    pq.add(new Info(v,arr[v],curr.k+1));
                }
            }
        }
        return arr;
    }
    public int findCheapestPrice(int n, int[][] flights, int src, int dest,int k) {
        ArrayList<Edge>[] graph=new ArrayList[n];
        for(int i=0;i<graph.length;i++)graph[i]=new ArrayList<>();
        createGraph(graph,flights);
        int[] arr=dijkstra(graph,src,k);
        return arr[dest]==Integer.MAX_VALUE ? -1 : arr[dest];
    }
}
